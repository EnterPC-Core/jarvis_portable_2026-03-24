# Runtime Backups

Здесь лежат GitHub-safe снимки runtime-состояния проекта.

Что входит:

- схемы SQLite-баз
- счётчики строк по таблицам
- безопасные служебные таблицы (`bot_meta`, `chat_modes`, `warn_settings`, `welcome_settings`)
- manifest последнего обновления

Что не входит:

- сырые переписки
- содержимое `chat_history`
- содержимое `chat_events`
- приватные сообщения
- полный дамп legacy-данных пользователей

Обновление:

```bash
python3 tools/export_runtime_backups.py
```

Эти файлы предназначены для синхронизации проекта и репозитория без публикации чувствительных данных.
